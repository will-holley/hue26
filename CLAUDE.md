# Project Instructions

## Package Manager

This project uses **pnpm** for package management. Always use `pnpm` commands instead of npm or yarn.

Examples:
- Install dependencies: `pnpm install`
- Add a package: `pnpm add <package-name>`
- Add a dev dependency: `pnpm add -D <package-name>`
- Run scripts: `pnpm run <script-name>` or `pnpm <script-name>`

## Project Structure

Source code is located in `/src`
